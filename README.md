# Four Corners
### To Run
Create a .env file with
* DB_USER
* DB_PASS
* DB_HOST
* DB_NAME

run a live server of lobby/lobby.html


run
```
npm install
node lobbyserver.js
```


run the next section of code from checkers, chess, connect4, and mancala directories
```
node index.js
```
